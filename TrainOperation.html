    운영

    <열차운행>
    - 코레일 운영시간표를 바탕으로 운행한다.

    - 새마을 임시열차 : 목포~광주송정
    - KTX, KTX-산천, KTX-이음, SRT, 해무 임시열차 : 목포~광주송정

    <도시 ＆ 이설 ＆ 폐지>
    - 도시 이전 : 목포
    - 역 이설 : 목포
    - 기존 역 삭제 : 목포
    - 기존도시 삭제 : 목포

    <역건설>
    - 광주지하철 : 평동차량기지~광주송정
    - 호남선 : 목포~광주송정 역건설

    <실제 위도, 경도 데이터 수집 완료>
    - 지하철 : 부산김해경전철(가야대~사상), 동해선(부전~일광), 대전지하철(판암~반석), 대구지하철 3호선(칠곡경대병원~용지), 대구지하철 2호선(문양~영남대), 대구지하철 1호선(안심~설화명곡), 광주지하철(평동~녹동)
    - 일반철도 : 호남선 (목포~서대전), 장항선 (익산~천안), 전라선 (익산~여수엑스포), 경전선 (광주송정~순천)

    <맵 세팅>
    맵크기 : 4096(동서) X 8192(남북)의 거리로 진행한다.
    바다가 없는 평지맵으로 진행하고, 열차가 이동하며 만나는 터널은 선로 건설을 진행하면서 별도로 만들며 진행한다.
